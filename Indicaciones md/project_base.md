# RenovaIQ — Base del Proyecto

## Objetivo
Calculadora web de estimación para remodelación de baños.

## Restricciones técnicas
- PC modesta: Celeron + 8GB RAM
- Debe correr ligero con npm run dev
- Evitar dependencias pesadas
- Priorizar simplicidad y estabilidad

## Stack permitido
- Next.js 14 App Router
- JavaScript
- React Hooks
- Tailwind CSS
- jsPDF + jspdf-autotable
- Formspree

## Stack prohibido
- TypeScript
- Backend propio
- Base de datos
- Supabase
- Stripe
- Librerías pesadas innecesarias

## Regla de diseño
- Mobile-first
- Pocas pantallas
- Código simple
- Nada que complique el rendimiento
